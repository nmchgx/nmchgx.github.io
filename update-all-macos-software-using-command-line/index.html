看名字应该能猜到 Homebrew-Cask （以下简称Cask）是 Homebrew 的衍生品，可用来管理非App Store下载的软件。

安装 Cask ：

有了 Cask 安装软件变得很简单，只需要 brew cask install 加上软件的名称即可。
